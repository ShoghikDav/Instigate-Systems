Մեր Մասին

«Ինստիգեյթ Սիսթեմս» ՓԲԸ-ն ծրագրավորման ընկերություն է

Ընկերությունը ծառայություններ է մատուցում համակարգչային ծրագրավորման ոլորտում

  • 2011 Մայիսի 20

    Ընկերության հիմնադրումը Ստեփանակերտում

    Արցախում հաջողությամբ գործունեություն է իրականացնում ծրագրային ապահովման մշակմամբ զբաղվող առաջին ընկերությունը` «Ինստիգեյթ Սիսթեմսը»: Հայաստանյան «Ինստիգեյթ» ՓԲԸ-ի բաժնետերերի ժողովը 2011թ ապրիլի 22-ին որոշեց Արցախում ստեղծել դուստր ընկերություն։ Մայիսի 20-ին դուստր ընկերությունը ստացավ պետական գրանցում` «Ինստիգեյթ Սիսթեմս» անվանումով։

  • 2010 Սեպտեմբեր

    Ընկերության հիմնադրման պատմությունից

    Մինչ դուստր ընկերության հիմնումը` 2010թ. սեպտեմբերին Ինստիգեյթ ՓԲԸ-ն հայտարարեց իրեն պատկանող Ինստիգեյթ ուսումնական կենտրոնի միջոցով Ստեփանակերտում թրեյնինգ կենտրոն բացելու մասին: Մեկ ամիս անց կենտրոնը սկսեց գործել և շուրջ մեկ տասնյակ արցախցի երիտասարդներ դարձան դրա ուկնդիրը: Արցախի պետական համալսարանի ֆիզիկամաթեմատիկական ֆակուլտետի ուսանողները և շրջանավարտները համակարգչային ծրագրավորման և ինժեներական պրակտիկայի դասընթացներ էին անցնում։Արդեն 2011թ. մայիսին նրանցից յոթը ստացան իրենց ավարտական վկայականները:

  • 2015

    Ընկերության Հաջողությունները

    Հիմնադրման օրից մինչ այսօր ընկերությունը հասել է մեծ հաջողությունների։ Ընկերության գործարկումից ի վեր ստեղծվել են բազմազան ֆունկցիոնալ հնարավորություննեով, յուրահատուկ դիզայնով վեբ կայքեր պետական և մասնավոր կազմակերպությունների համար, որոնք գործում են տարբեր շուկաներում ու ոլորտներում։ Այս տարիների ընթացքում իրեն դրսևորել է որպես հուսալի գործընկեր արտասահմանյան և տեղական ընկերությունների համար։

Մեր փորձը որպես մոբայլ և վեբ ծրագրերի ծառայություններ մատուցող ընկերություն

ՎԵԲ ԾՐԱԳՐԱՎՈՐՈՒՄ

Նոր կայքի ձևավորում և ծրագրավորում: Առկա կայքի ձևավորման փոփոխում: Առկա կայքի վրա ծրագրային փոփոխում կամ ավելացում։ Օգտագործվող տեխնոլոգիաներ՝ HTML 5, CSS 3, JavaScript, jQuery, Ajax, Bootstrap, PHP, MySQL

ՄՈԲԱՅԼ ՀԱՎԵԼՎԱԾՆԵՐ

Մոբայլ հավելվածների ստեղծումն այսօր ամենատարածված ոլորտներից մեկն է: Այն շատ մեծ տարածում գտավ հատկապես IOS և Android համակարգերի շուկան գրավելուց հետո: Օգտագործվող տեխնոլոգիաներ` Android, Windows Mobile միջավայրում՝ SenchaTouch, JqueryMobile, Java

ՈՐԱԿԻ ԱՊԱՀՈՎՈՒՄ

Կայքի վեբ ծրագրավորումը ավարտելուց հետո, այն անցնում է մանրակրկիտ թեստավորման: Հայտնաբերված թերությունները վերացվում են անմիջապես: Օգտագործվող տեխնոլոգիաներ` ծառայությունների թեստավորում Agile պրոցեսի կիրառմամբ և ավտոմատացում Robot Framework, Selenium, Python, թեստավորման գործընթացը կառավարող ծրագրեր՝ JIRA, Redmine
